Описание проекта

Этот мини-сайт состоит из трёх взаимосвязанных страниц
и представляет собой небольшой интерактивный опыт:
от мрачного символа HU до финального мира Sakura Paradise 🌸

📂 Структура проекта
HU_Site/
├── index.html     ← первая страница (HU + дверь)
├── page2.html     ← терминал с таймером и паролем
├── page3.html     ← розовая Sakura Paradise
└── README.txt      ← этот файл-инструкция

🚀 Запуск

Способ 1 — через Visual Studio Code + Live Server

Установи VS Code: https://code.visualstudio.com/

Установи расширение Live Server (Ritwick Dey).

Открой папку HU_Site в VS Code.

Открой файл index.html.

Нажми Go Live (внизу справа).

Браузер откроет:

http://127.0.0.1:5500/index.html


⚠️ При открытии напрямую (двойным кликом) браузер может блокировать музыку и переходы.

🎮 Навигация по страницам

index.html

Чёрный экран, две буквы HU (пульсирующие).

В правом углу — дверь.

После 13 кликов дверь открывается и ведёт на следующую страницу.

Внизу — значок письма (email).

page2.html

Появляется предупреждающий текст.

Активные чекбоксы (все включены → пауза таймера).

Таймер обратного отсчёта (13 секунд).

Пароль: 28031975

Верный → переход к Sakura Paradise.

Неверный → звуковой сигнал и возврат на первую страницу.

page3.html

Розовый градиент, плавающие котики 🐈 и цветы 🌸.

Падающие лепестки.

Кнопка Music On/Off включает музыку.

Кнопка-лепесток Back to Start возвращает на первую страницу.

При клике — надпись “Returning to HU…” и световой эффект.

🛠 Редактирование

Открой любой .html файл в VS Code.

Измени текст, цвета, музыку, эффекты.

Сохрани (Ctrl + S).

Обнови страницу (Ctrl + R).

⚙️ Примечания

Лучше тестировать проект через Live Server — все переходы и анимации будут работать корректно.

Автомузыка блокируется политикой браузеров, поэтому звук включается вручную.

Все изображения встроены в код в формате SVG data URI — проект автономен и не требует интернета.
